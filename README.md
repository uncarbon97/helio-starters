# helio-starters

## 项目介绍

`helio-starters`是Helio脚手架及其业务微服务模块的基础支撑构件，可以快速引入其他中间件和基础配置

JDK compatibility: 1.8 - 17

[官方文档](https://helio.uncarbon.cc/)
[主要依赖](https://helio.uncarbon.cc/#/i18n/zh-CN/helio-starters/dependencies)
